# 游戏规则

1. 滑块在不停向下滚动，手指触摸点击滑块。
2. 若点击到黑滑块，黑滑块变绿，游戏继续。
3. 若点击到白滑块，白滑块变红，游戏结束。
4. 若在黑滑块消失之前没有变绿，游戏结束。
